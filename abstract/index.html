Our Atlassian Setup

Portfolio

Manages our long term product roadmap and reports on our progress. Results in realistic and transparent timelines.

JIRA Software

Manages our day-to-day execution. Kanban Boards (no sprints). Results in an organized workflow.

HipChat

Eliminates excess meetings. Facilitates faster decision making. Results in higher productivity.

Custom Integrations

Issues are transitioned by events in GitHub. GitHooks result in accurate and up-to-date timesheets.

Portfolio

Each project generally has 3 epics that roll up into 1 initiative
Weekly reviews of Portfolio keep management in the loop on progress
Quarterly scoping excercises allow us to forecast and run hiring scenarios

JIRA Software (Kanban)

Three Main Kanban Boards host the day-to-day tasks for Design, Front End, and Back End teams as well as their Portfolio Epics (no children)
Each Epic gets its own Kanban board to host all of its children (tasks and bugs). Once the board is cleared, the project is ready to launch.
Source Control Integration via DVCS Connection keeps our code and our issues in sync. Our boards are always up to date with real-time status.

HipChat

Primary room for each team allows us to coordinate on incoming requests, and effectively use each other as knowledge resources
Integrations for deploy notifications, issue updates, system alerts, and code changes keeps everyone in the loop on real-time changes
Rooms replace face-to-face meetings when teammates need quick decisions while multitasking, or if no conference rooms are available.

Custom Integrations

GitHub events are relayed to JIRA with a custom integration for faster transition of issues. Custom events such as labelling Pull Requests, or synchronizing a pull reqeuest will transition issues with the custom integration
Integrations for deploy notifications, issue updates, system alerts, and code changes keeps everyone in the loop on real-time changes
Githook for logging time results in more accurate timesheets with less effort. Our timesheets are always up to date which is big for us since our Portfolio plan relies on time spent when reporting on progress.
